ZF 1 - Настройка Zend Tool в Ubuntu

2012-08-14
Zend Framework

Zend Tool предоставляет возможность работать с проектом на Zend Framework используя терминал. Существуют команды для создания проекта, контроллеров, модулей, моделей и пр. Он удобен тем, что освобождает программиста от рутинной работы. Например, при создании проекта создается каркас будущего приложения и его начальная конфигурация. При создании контроллеров, так же создается каркас контроллера с базовым действием IndexAction и видом к нему. Это удобней, чем каждый раз проделывать одни и те же действия вручную.

Для того что бы настроить Zend Tool в Ubuntu, нужно проделать несколько простых действий.

  1. Идем на http://framework.zend.com/ и скачиваем Zend Framework.

  2. Далее распаковываем Zend Framework в домашнюю папку, структура должна получиться следующей:

  1. Для того что бы команда "zf" была доступна из любой директории, нужно прописать к ней алиас. Для этого идем в домашнюю папку, открываем файл .bashrc и в его конец добавляем следующее:
alias zf=/home/username/ZendFramework/bin/zf.sh

Где "username" замените на имя своего пользователя в Ubuntu.

Примечание: Файл .bashrc является скрытым и не отображается Наутилусом. Что бы увидеть его, нажмите Ctrl + H.

  1. Теперь нужно завершить текущий сеанс в Ubuntu.

  2. И напоследок проверка, открываем терминал и выполняем команду:
zf show version

Должно отобразиться примерно следующее:

Zend Framework Version: 1.11.12
