ZF 1 - Интеграция c NetBeans

2012-12-15
Zend Framework, NetBeans

Zend Framework предоставляет удобный инструмент Zend Tool, предназначенный для быстрого создания проекта, контроллеров, моделей и пр. элементов приложения. В NetBeans есть возможность настроить взаимодействие с Zend Tool. В данной статье речь пойдет о настройке интеграции Zend Tool с NetBeans, в ОС Ubuntu.

Интеграция Zend Tool с NetBeans

Первым делом необходимо скачать Zend Framework и настроить работу Zend Tool в консольном режиме. О том как это сделать вы можете прочитать в предыдущей статье Настройка Zend Tool в Ubuntu.

После того как вы настроили Zend Tool в консольном режиме, можно приступать к интеграции его с NetBeans. Для этого открываем параметры среды NetBeans, меню Сервис -> Параметры. В окне параметров переходим в раздел PHP, на вкладку Zend.

На данной вкладке нажимаем на кнопку "Обзор" и выбираем сценарий zf.sh. Далее нажимаем на кнопку "Зарегистрировать поставщика", если все прошло успешно, отобразится уведомление следующего вида:

Provider/Manifest 'NetBeansCommandsProvider' was enabled for usage with Zend Tool.

После этого в окне настроек NetBeans нажимаем на кнопку OK.

После проделанных манипуляций, нам станут доступны две удобные вещи касающиеся Zend Framework. Первая - это создание каркаса приложения на Zend Framework прямо из среды NetBeans, без необходимости набирать команду в консоле. И вторая - возможность выполнять команды Zend Tool так же из среды NetBeans.

Создание каркаса приложения на Zend Framework из среды NetBeans

Делается это просто, создаем новый проект, выбираем PHP -> Приложение PHP, жмем на кнопку "Далее".

Далее выбираем расположение проекта, указываем его имя. На последнем этапе ставим галку на "Веб-платформа Zend PHP" и нажимаем на кнопку "Готово".

После этих действий автоматически создастся каркас приложения на Zend Framework.

Выполнение команд Zend Tool из среды NetBeans

После того как вы создали новый проект на Zend Framework, возникает необходимость наращивать его функциональность, создавая контроллеры, модели и прочие элементы. Для этого в среде NetBeans, вызываем контекстное меню на созданном проекте и выбираем в нем Zend -> Выполнить команду.

Появится окно для выполнения команд Zend Tool, причем команды можно как набирать вручную, так и выбирать из списка с подсказками, что на мой взгляд очень удобно.

Но согласитесь, не удобно каждый раз вызывать в меню Zend -> Выполнить команду окно для выполнения команд Zend Tool. Поэтому следующим шагом мы настроим сочетание клавиш для быстрого вызова данного окна. Открываем параметры среды NetBeans в меню Сервис -> Параметры. В окне параметров переходим в раздел "Назначения клавиш", в поле "Поиск" вводим "zend", должен появится пункт "Zend: Выполнить команду". Задаем нужное сочетание клавиш, например Ctrl+Alt+Z и нажимаем на кнопку OK.

После этого, нажав Ctrl+Alt+Z в редакторе, будет появляться окно для выполнения команд Zend Tool.
