Vagrant / VirtualBox - Failed to create the host-only adapter

2016-08-04
VirtualBox, Vagrant

When using Vagrant on Windows 10 for managing local virtual machine, began to occur an error: Failed to create the host-only adapter. Further on how to fix it.

We have: Windows 10 / Vagrant 1.8.5 / VirtualBox 5.1.2

When run the command vagrant up occurs error just about the following look:

Command: ["hostonlyif", "create"]

Stderr: 0%...
Progress state: E_FAIL
VBoxManage.exe: error: Failed to create the host-only adapter
VBoxManage.exe: error: Code E_FAIL (0x80004005) - Unspecified error (extended info not available)
VBoxManage.exe: error: Context: "int __cdecl handleCreate(struct HandlerArg *,int,int *)" at line 66 of file VBoxManageHostonly.cpp

And earlier, on Windows 8 this error was not.

As it turned out, in VirtualBox have already taken care of this and released a special exe-fix, which bring VirtualBox to feelings:

  1. Yo need go to the next link: https://www.virtualbox.org/attachment/ticket/14040/VBox-Win10-fix-14040.exe.
  2. On the opened page click to "downloading" link.
  3. After exe file has been downloaded, run it with an administrator rights.
  4. Try run the command vagrant up, must have no errors.
  5. After those actions you can delete downloaded exe file.
