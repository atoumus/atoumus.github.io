Objective-J - Подсветка синтаксиса в Sublime Text

2012-09-22
Objective-J, Sublime Text

Sublime Text позволяет установить плагин, для подсветки синтаксиса языка Objective-J (Cappuccino). В этой небольшой заметке, пойдет речь о том, как это сделать.

Первым делом необходимо установить Sublime Package Control. Для этого запустите ваш Sublime Text и нажмите сочетание клавиш: Ctrl+`, должна открыться консоль в нижней части редактора.

Далее переходим по этой ссылке, откроется страница проекта Sublime Package Control. На этой странице будет указана команда, которую нужно скопировать, вставить в консоль в Sublime Text и нажать Enter. После того как Package Control будет установлен, перезапустите Sublime Text.

Теперь установим плагин, который добавить в ваш Sublime Text подсветку синтаксиса языка Objective-J. Для этого запустим только что установленный Package Control. Это можно сделать из меню Preferences -> Package Control или с помощью сочетания клавиш Ctrl+Shift+P.

В открывшемся Package Control набираем слово install и выбираем появившейся пункт меню Install Package. Откроется список пакетов доступных для установки. Набираем слово cappuccino и выбираем Cappuccino bundle for sublime text 2. После того как установка будет завершена, в строке статуса появиться соответствующее уведомление.

Далее, открываем нужный .j файл, выбираем в меню View -> Syntax -> Cappuccino -> Objective-J и получаем качественную подсветку синтаксиса + автодополнение ключевых слов языка.
